 Resumo do Projeto para GitHub
🎯 Por que este projeto existe?
Este projeto foi criado para demonstrar habilidades práticas em desenvolvimento web front-end através de um site institucional completo e funcional.
💡 O Problema que Resolve
Muitos cursos de programação ensinam teoria, mas faltam projetos reais que mostrem como aplicar HTML, CSS e JavaScript juntos em um produto profissional. Este projeto preenche essa lacuna ao criar um site completo do zero, sem frameworks, mostrando que é possível construir interfaces modernas e interativas com tecnologias puras.
Decisões de Design
Por que HTML/CSS/JS puro?
✅ Performance máxima — sem overhead de frameworks
✅ Controle total — cada pixel está onde deve estar
✅ Base sólida — entender o fundamental antes de usar abstrações
✅ Portfólio impressionante — mostra domínio real das tecnologias web
Por que essa paleta de cores?
Verde (#4ADE80) — representa crescimento, tecnologia e sucesso
Roxo (#6B21A8) — transmite criatividade, inovação e sofisticação
Dark mode — moderno, reduz fadiga visual, padrão da indústria tech
Por que essas animações?
Reveal on scroll — guia o olhar do usuário
Cards flutuantes — adiciona profundidade e vida
Microinterações — feedback visual em cada ação
Transições suaves — experiência premium
